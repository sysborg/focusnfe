# Contributing

## Objetivo

Este projeto busca oferecer uma integracao Laravel com a FocusNFe que seja previsivel, bem documentada e segura para producao.

## Fluxo recomendado

1. Crie uma branch a partir da branch principal.
2. Faça mudancas pequenas e focadas.
3. Adicione ou atualize testes sempre que alterar comportamento.
4. Rode, se disponivel no seu ambiente, lint, analise estatica e testes.
5. Abra um pull request descrevendo contexto, mudancas e riscos.

## Padroes esperados

- PSR-12
- Laravel Pint
- PHPStan/Larastan
- PHPUnit

## O que incluir em um PR

- motivacao da mudanca
- impacto esperado
- arquivos afetados
- cobertura de testes adicionada ou atualizada
- notas de migracao, se houver

## Tipos de contribuicao bem-vindos

- correcoes de bugs
- testes faltantes
- documentacao
- novos endpoints suportados pela API
- melhorias de DX sem quebrar compatibilidade publica
